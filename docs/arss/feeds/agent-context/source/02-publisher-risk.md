# Publisher risk in the agent web

Publishers are exposed by the agent layer because agents can consume the value of content without creating the old economic event: a page view, ad impression, session or subscription login.

Robots.txt is too crude for this world. A publisher needs to express: this agent may summarise, this content may be cached for seven days, quotations are capped at 500 characters, model training is prohibited, full text costs one cent via x402, and attribution is required.

ARSS gives publishers a direct agent-subscription surface instead of forcing them to choose between blocking agents and being scraped.
