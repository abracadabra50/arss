# ARSS paper

LaTeX source for **ARSS: Agent RSS for Paid Context Feeds**.

## Build

```bash
make
```

Uses the local standalone Tectonic binary at `../../../tools/tectonic/tectonic` by default. Override with `make TECTONIC=tectonic` if you have it on PATH.

Outputs:

- `arss-agent-rss-paper.pdf`
- `build/main.log`
